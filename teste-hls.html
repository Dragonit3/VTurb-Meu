🔍 Como Funciona:

  1. Carregamento: O script baixa o HLS.js automaticamente
  2. Detecção: Verifica se o navegador suporta HLS nativo (Safari) ou via HLS.js
  3. Streaming: Carrega o vídeo em qualidade adaptativa
  4. Analytics: Registra impressão, play, pause, fim do vídeo
  5. Visual: Barra de progresso customizada e loading indicator

📈 Métricas Coletadas:

  • 🎯 Impressão: Quando o player é carregado
  • ▶️ Play: Primeiro clique em play (com dados do navegador)
  • ⏸️ Pause: Quando o vídeo é pausado
  • 🏁 Fim: Quando o vídeo termina (com % de conclusão)
  • 🌐 Contexto: URL, referrer, user agent, viewport
